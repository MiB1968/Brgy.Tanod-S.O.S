# Contributing to Brgy Tanod S.O.S.

Thank you for considering contributing to this project! This system aims to help save lives in Philippine barangays.

## Development Setup

```bash
git clone https://github.com/MiB1968/Brgy.Tanod-S.O.S.git
cd Brgy.Tanod-S.O.S
npm install
npm run dev
```

For Firebase Functions:
```bash
cd functions
npm install
npm run build
```

## Code Guidelines

- Use **TypeScript** strictly
- Follow functional component pattern in React
- Use Singleton pattern for services
- Keep components under 300 lines when possible
- Write meaningful commit messages

## Pull Request Process

1. Fork the repository
2. Create a feature branch (`feature/amazing-feature`)
3. Make your changes
4. Test thoroughly (especially offline mode)
5. Submit a Pull Request with clear description

## Areas Needing Help

- Unit & Integration tests
- iOS Capacitor optimization
- Tagalog AI model fine-tuning
- UI/UX improvements
- Documentation & video tutorials

## Code of Conduct

Be respectful. This project is meant to serve public safety.
